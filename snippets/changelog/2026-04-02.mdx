### Product changes

- **`X-Generation-Id` response header** -- All inference endpoints now return an `X-Generation-Id` header for easier request tracing and debugging.
- **Output modality tabs** -- The [Models](/guides/overview/models) page now uses a tab-based filter for output modalities, replacing the previous sidebar dropdown.
- **Chatroom improvements** -- The chatroom now displays dates, real author names, and includes a datetime tool.
- **Tool error messages** -- Tool compatibility errors now include the first tool name in the message for easier debugging.
