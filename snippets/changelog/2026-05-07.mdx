### Product changes

- **Provider-specific image generation quickstarts** -- Model pages for Recraft V3, Recraft V4 / V4 Pro, and Sourceful now show tailored code examples with each model's supported `image_config` parameters (e.g., `text_layout` for V3, `rgb_colors` for V4, `font_inputs` for Sourceful). See the [image generation docs](/guides/overview/multimodal/image-generation).
- **Activity page API key labels** -- Activity analytics now correctly display API key names instead of numeric IDs for accounts with more than 500 API keys. Previously, the slim-keys endpoint capped at 500 results, leaving excess keys unlabeled in charts.
