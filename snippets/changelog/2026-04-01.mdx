### Product changes

- **Indian e-mandate payments** -- Added support for Indian e-mandate payments, enabling recurring off-session billing for users with Indian bank cards.
- **Design Arena benchmarks** -- Model detail pages now display [Design Arena](https://designarena.ai) benchmark rankings with bar chart visualizations.
- **Fusion (beta)** -- [Fusion](https://openrouter.ai/fusion) is now accessible from the navigation bar, letting you run prompts across multiple models simultaneously.
- **Uptime page improvements** -- The [Uptime](/guides/best-practices/uptime-optimization) page now has a sort dropdown and improved tick filling for better readability.
- **Generation ID search** -- Searching by generation ID in activity logs now jumps directly to the detail pane.
- **`allowed_tools` in tool_choice** -- The Responses API now supports `allowed_tools` in `tool_choice`, letting you restrict which tools the model can call.
- **Uptime in endpoints API** -- `uptime_last_5m` and `uptime_last_1d` fields are now available in the public endpoints API.
