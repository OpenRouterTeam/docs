### Product changes

- **Speech-to-Text (Transcription) models added** -- A `/api/v1/audio/transcriptions` endpoint is now available along with SDK support. See the [Transcription docs](/guides/overview/multimodal/audio) for details.
- **Speech-to-text quickstart examples** -- Added transcription code samples across all supported SDK languages in the [Quickstart](/quickstart).
- **Guardrail indicator on logs** -- The [Activity logs](https://openrouter.ai/activity) table now shows which requests triggered a guardrail.
- **Copy BYOK provider key to workspace** -- You can now duplicate a provider key into a workspace from the BYOK settings.
- **API key detail sheet** -- Clicking an API key now opens a detail sheet with usage stats and metadata.
- **Web tool URL preview in Playground** -- The [Playground](https://openrouter.ai/playground) now surfaces the fetched URL for web search tool calls and expands details on click.
- **Nitro variant throughput sorting** -- The [Pareto router](/guides/routing/routers/pareto-router) now sorts `:nitro` variant model selection by throughput for faster inference.
