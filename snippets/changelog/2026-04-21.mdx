### Product changes

- **Throughput metrics** -- Added tokens/second throughput metrics to the [Activity Explorer](https://openrouter.ai/activity).
- **View Logs shortcut** -- Activity page filters now include a "View Logs" link that passes your current filters through to the logs view.
- **Date range filter options** -- Added "Past hour" and "Past 24 hours" quick filter options to the date range picker.
- **Guardrails on API key detail** -- The API key detail page now shows guardrail configuration.
- **Logs refresh indicator** -- The refresh button on the [Activity](https://openrouter.ai/activity) page now shows a spinning icon while loading.
- **`web_fetch` server tool** -- Re-added the `openrouter:web_fetch` server tool for fetching web content during inference.
