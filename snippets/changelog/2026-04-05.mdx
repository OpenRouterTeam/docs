### Product changes

- **RAG guide** -- Published a new guide on building [Retrieval-Augmented Generation](/cookbook/evaluate-and-optimize/rag) pipelines using embeddings and rerank.
- **App slug-based URLs** -- Apps now support custom slug-based URLs for cleaner, shareable links to app pages.
- **Models API `links` field** -- The models API response now includes a `links` field with relevant external references for each model.
- **Rerank and embedding model pages** -- Chat button is now hidden on embedding and rerank model pages since these modalities don't support chat.
