### Product changes

- **Claude Desktop integration guide** -- New guide for integrating [Claude Desktop with OpenRouter](/guides/community/frameworks-and-integrations-overview).
- **Text-to-Speech documentation** -- Added a dedicated [Text-to-Speech documentation page](/guides/overview/multimodal/tts).
- **Cache hit indicator on activity log** -- The [Activity](https://openrouter.ai/activity) log now shows a cache hit indicator and filter.
- **Load more pagination** -- Replaced traditional pagination with a "Load more" button across all [Activity](https://openrouter.ai/activity) tabs.
- **Video webhook settings** -- Video webhook settings are now public with `callback_url` in the request schema. See the [webhook docs](/guides/features/broadcast/webhook).
- **Webhook docs update** -- Simplified the webhook signature format description in the [webhook docs](/guides/features/broadcast/webhook).
- **Requests chart rename** -- The "Prompts" chart is now labeled "Requests" with longest streak tracking.
- **Dynamic model cards** -- Model card heights are now dynamic instead of fixed for better content display.
- **Logs timezone toggle fix** -- The UTC/Local timezone toggle now correctly updates table timestamps.
- **Mobile generation details** -- Fixed discount display and close button on mobile generation details view.
