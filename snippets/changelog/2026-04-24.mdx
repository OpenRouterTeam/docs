### Product changes

- **Compare page redesign** -- New breadcrumbs, preset pills, layout, suggestions, and inline API link on the [Compare](https://openrouter.ai/models/compare) page.
- **Guardrails column on API keys** -- API keys list view now shows which guardrails are attached to each key.
- **SDK docs reorganization** -- SDK documentation split into separate [Client SDKs](/client-sdks/overview) and [Agent SDK](/agent-sdk) tabs.
- **Agent TUI guide** -- New guide for [building your own agent TUI](/guides/community/frameworks-and-integrations-overview).
- **Theme: system default** -- OpenRouter now defaults to the system theme, respecting your OS dark/light preference.
- **DeepSeek V4 tool history fix** -- Fixed native tool history preservation for DeepSeek V4 thinking + tools.
- **Chatroom: audio model filter** -- Audio output models are now filtered from the chatroom model selector.
- **Duplicate SSE error chunk fix** -- Fixed duplicate terminal error chunks when upstream providers send multiple error SSEs.
