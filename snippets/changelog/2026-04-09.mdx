### Product changes

- **Web search upgrade** -- Web search now uses improved text content retrieval for more accurate and relevant results.
- **Faster session loading** -- Session sidebar and tab switching now use optimistic state updates for immediate visual feedback.
- **Overloaded error fallback** -- The router now buffers `message_start` events to enable provider fallback when an `overloaded_error` occurs mid-stream.
- **Playground state preserved across login** -- Playground prompt text and attachments on model pages are now preserved when redirecting through login.
- **Video generation docs** -- Published comprehensive [video generation documentation](/guides/overview/multimodal/video-generation) covering endpoints, parameters, and async generation.
- **User activity heatmap** -- A new activity heatmap visualization is available on user profile pages.
