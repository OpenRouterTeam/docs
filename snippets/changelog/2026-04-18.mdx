### Product changes

- **Text-to-speech launch** -- Text-to-speech is now available on OpenRouter with support for multiple providers including OpenAI, ElevenLabs, Cartesia, Google, Mistral, and xAI. See the [text-to-speech docs](/guides/overview/multimodal/tts).
- **Text-to-speech playground** -- A dedicated [text-to-speech tab](https://openrouter.ai/playground) in the Playground lets you test speech synthesis with voice selection and audio playback.
- **Models page redesign** -- The [Models](/guides/overview/models) page now features author icons, modality badges, tab highlights, and an improved card layout.
- **Video playground uploads** -- Added frame image and reference image upload UI to the video generation [Playground](https://openrouter.ai/playground).
- **Provider applications** -- Providers can now apply to join OpenRouter through a new [provider application page](https://openrouter.ai/providers/apply).
- **Tokens metric for embeddings** -- Embedding models now show a Tokens metric option on the [Activity](https://openrouter.ai/activity) page.
