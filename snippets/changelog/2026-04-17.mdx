### Product changes

- **Workspaces** -- Workspaces are now available, allowing teams to organize API keys, guardrails, and usage under separate environments within an organization. See the [Workspaces docs](/guides/features/workspaces).
- **Privacy settings page** -- New account-level [privacy settings](https://openrouter.ai/settings/privacy) page for managing data preferences.
- **Get Code button in playground** -- The [Playground](https://openrouter.ai/playground) now has a "Get Code" button that generates ready-to-use code snippets for your current configuration.
- **SDK examples on API tab** -- Added OpenRouter Python/Go and Anthropic TS/Go SDK examples to the model page API tab, grouped by SDK category with sub-tabs.
- **Session timestamps** -- The sessions table now displays timestamps alongside dates for more precise session tracking.
