### Product changes

- **Cmd+K search shortcut** -- Press `Cmd+K` (or `Ctrl+K`) anywhere on the site to open the global search.
- **Guardrail eligibility chain viewer** -- API key inspector now shows the full guardrail eligibility chain for each key.
- **Audio input rankings** -- The [Rankings](https://openrouter.ai/rankings) page now includes an audio input section.
- **Cache hit token breadcrumbs** -- Dashboard savings widgets now display token breadcrumbs on cache hits.
- **Stripe Projects blog post** -- Published a blog post about the [Stripe Projects integration](https://openrouter.ai/announcements/stripe-projects).
- **Models page scroll performance** -- Improved scroll performance and fixed iOS Safari scroll momentum on the [Models](/guides/overview/models) list view.
