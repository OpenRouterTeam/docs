### Product changes

- **Announcements RSS feed** -- Subscribe to OpenRouter announcements via RSS at [/announcements/feed.xml](https://openrouter.ai/announcements/feed.xml).
- **Usage response: cost & BYOK fields** -- API responses now include `cost`, `is_byok`, and `cost_details` in the `usage` object. [API Reference](/api/reference)
- **Workspace analytics chart** -- Workspace overview now shows a stacked bar chart of usage by model.
- **Logs: "Live" time window** -- Renamed "Tail" to "Live" on the logs page time window picker for clarity.
- **Responses API: unknown tool passthrough** -- The Responses API no longer silently strips unknown tool types; they are passed through to the model.
- **Codex CLI compatibility fix** -- Fixed mixed Chat Completions / Responses tool shapes to unbreak Codex CLI 0.125+.
- **`~latest` enrichment** -- `~latest` models (e.g. `gpt-4o~latest`) are now enriched with full metadata on the [Models](/guides/overview/models) page.
- **iOS Safari search auto-zoom fix** -- Fixed auto-zoom on search input focus on iOS Safari.
- **Logs wide viewport columns** -- Log table columns now extend properly across wide viewports.
