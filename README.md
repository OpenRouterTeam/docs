# OpenRouter Docs

This repository is the public issue tracker for [OpenRouter's documentation](https://openrouter.ai/docs) and API. Use it to report bugs, suggest improvements, or flag inaccuracies.

## Quick Links

- [Documentation](https://openrouter.ai/docs)
- [API Reference](https://openrouter.ai/docs/api-reference)
- [Quickstart](https://openrouter.ai/docs/quickstart)
- [Status Page](https://status.openrouter.ai)

## Reporting Issues

Found a problem? [Open an issue](https://github.com/OpenRouterTeam/docs/issues/new/choose) and pick the template that fits:

- **Docs Bug** -- Incorrect information, broken links, typos, or missing content on [openrouter.ai/docs](https://openrouter.ai/docs)
- **API Bug** -- API behavior that doesn't match what the docs describe (wrong response shapes, missing parameters, endpoint discrepancies)

For general questions, head to [Discord](https://discord.gg/fVyRaUDgxW) instead.

## SDKs

For issues specific to an SDK or its documentation, file them in the appropriate repo:

- [TypeScript SDK](https://github.com/OpenRouterTeam/typescript-sdk)
- [Python SDK](https://github.com/OpenRouterTeam/python-sdk)
