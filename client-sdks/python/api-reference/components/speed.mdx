Controls output generation speed. When set to `fast`, uses a higher-speed inference configuration at premium pricing. Defaults to `standard` when omitted.

## Values

| Name       | Value      |
| ---------- | ---------- |
| `FAST`     | fast       |
| `STANDARD` | standard   |
