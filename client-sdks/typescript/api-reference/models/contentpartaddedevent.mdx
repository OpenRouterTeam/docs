## Example Usage

```typescript lines
import { ContentPartAddedEvent } from "@openrouter/sdk/models";

let value: ContentPartAddedEvent = {
  contentIndex: 286298,
  itemId: "<id>",
  outputIndex: 549137,
  part: {
    text: "The capital of France is Paris.",
    type: "output_text",
  },
  sequenceNumber: 0,
  type: "response.content_part.added",
};
```
