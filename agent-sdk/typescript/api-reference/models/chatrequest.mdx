## Example Usage

```typescript lines
import { ChatRequest } from "@openrouter/sdk/models";

let value: ChatRequest = {
  messages: [
    {
      content: "You are a helpful assistant.",
      role: "system",
    },
    {
      content: "What is the capital of France?",
      role: "user",
    },
  ],
};
```
